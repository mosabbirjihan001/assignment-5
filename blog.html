Important FAQ

What is Document Object model (DOM) ?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of an HTML or XML document as a tree of objects that can be manipulated using JavaScript or other programming languages. The DOM allows developers to interact with the content, structure, and styles of a web page dynamically.

How do you select element from DOM ?

To select elements from the DOM, you can use several methods in JavaScript. The getElementById() method allows you to select an element by its unique id. For selecting multiple elements with the same class, you can use getElementsByClassName(), which returns a collection of matching elements. What is event delegation in DOM and why it is useful ?

Event delegation is a technique in JavaScript that allows you to manage events efficiently by attaching a single event listener to a parent element instead of multiple listeners to individual child elements. This is particularly useful when you have a large number of similar elements that need to respond to the same event. How Event Delegation Works: When an event occurs on a child element, the event bubbles up through the DOM tree to its parent elements. By attaching a single event listener to the parent, you can handle events for all child elements in one place.
